- Early Return과 else if의 연관관계
- else if는 조건의 순서를 잘 신경써야 한다.
- if문을 단독으로 사용하는 방식과 else if를 사용하는 방식은 요구사항에 따라 다르다. 상황에 맞는 적절한 선택이 필요하다.
- switch
  - switch vs. Java 14 switch
  - default의 break은 항상 생략하나요?
  - default문은 선택사항인가요? 필수로 작성해야 하는 상황은 없나요?
  - switch문은 비교연산자를 사용할 수 없나요?
- 삼항 연산자는 간결한 코드 작성에 사용한다. 코딩 테스트에서 효과적이다.
- if문으로 모든 조건을 처리할 수 있다. switch, 삼항 연산자는 if문을 더욱 간편하게 다룰 수 있도록 고안했다.
