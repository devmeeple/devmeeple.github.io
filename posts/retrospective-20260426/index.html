들어가며
#

1주차(3/29 ~ 4/4) - 시간아 멈춰
#

가끔 변하지 않는 것들이 있어 억만장자여도 이건 살 수 없어

해냈다
#

  • 김영한 ‘실전! 스프링 부트와 JPA 활용1 - 웹 애플리케이션 개발’ 완강
  • 『프로그래머의 길 멘토에게 묻다』(데이브 후버·애디웨일 오시나이, 인사이트, 2010) 완독
  • 『이 시대의 사랑』(최승자, 문학과지성사, 1981) 완독

느낀점
#

  • 알고리즘이 처음으로 아름답게 느껴졌다. 투포인터가 HashMap으로 표현되는 순간의 쾌감은 단순한 문제 풀이가 아니었다. 코드가 수학적 사고와 맞닿는 지점이 신기했다.
  • 데이터 베이스 지식이 크게 뚫려 있다는 걸 직시했다. JPA 강의를 따라가면서도 개념이 흐릿한 구간이 계속 나타났다. 매워야 할 곳이 보여 다음 방향성을 잡았다.
  • 목표가 흐릿해지자 모든 걸 서두르게 됐다. 여유가 없을 때 운동으로 균형을 맞췄다. 최승자 씨의 시처럼 고통 없는 행복은 없다. 지나온 시간은 밑거름이 될 것이다.

2주차(4/5 ~ 04/11) - 기쁨, 슬픔, 아름다운 마음
#

기쁨 뒤에 슬픔이 오는 건 아름다운 마음이야 쫒아내지 말고 품어주어라 아주 예쁜 돌이 된단다

해냈다
#

  • 남산둘레길 북측순환로 6.8km 완주
  • 황현우 ‘개발자를 위한 쉬운 리눅스’를 완강
  • 『숨』 완독
  • 『불온한 검은 피』 완독
  • 오피셜히게단디즘 라이브 앳 스타디움 2025 관람
  • 보드게임 페스타 참여

느꼈다
#

  • 『스프링 부트 개발자 온보딩 가이드: 스프링 부트로 시작하는 첫 실무 프로젝트를 통해 스프링 버전이 바뀌면 신경 써야 하는 점이 많다는 것을 몸소 배웠다. 데이터베이스 지식의 필용성을 크게 배웠으며 직렬화, 스프링 시큐리티가 궁금하다.
  • 다운힐에서 크게 절규했다. 10km 경험이 있으니 괜찮겠다 싶었던 자신감이 북측순환로 앞에서 산산이 부서졌다. 그래도 계속 도전하고 싶은 코스다. 고통스러웠던 것들이 자꾸 당기는 이유는 모르겠다.
  • 히게단 공연에서 ‘당신의 삶은 누군가의 원동력이 된다’는 메시지를 받았다. 자막이 있었지만 언어의 벽을 허무는 공연으로 느껴졌다.
  • 충동적인 소비를 하지 않아 다행이다. 보드게임 페스타에서 손에 쥐었다가 내려놓은 상자가 몇 개였는지, 잘 참았다.

3주차(04/12 ~ 04/18) - 群青
#

嗚呼、手を伸ばせば伸ばすほどに 遠くへゆく
思うようにいかない、今日も また慌ただしくもがいてる

해냈다
#

  • 김영한 ‘김영한의 실전 데이터베이스 입문 - 모든 IT인을 위한 SQL 첫걸음(SQL부터 차근차근)’ 완강
  • 『혼자 공부하는 컴퓨터 구조+운영체제』(강민철, 한빛미디어, 2022) 완독
  • 푸에르토리코 1897 플레이

느꼈다
#

  • SQL과 데이터베이스에 대한 이해가 부족하다고 느낀 시점에 김영한 님의 강의를 들어 좋았다. 바로 이어 진행되는 기본편이 기대된다.
  • 분명 고등학생 때 배운 컴퓨터구조+운영체제인데 낯설게 느껴졌다. 외계어로 느낀 부분이 더욱 많았다. 반복해서 읽으면 어떤 결과가 찾아올지 궁금하다.
  • 보드게임을 즐겼다. 평소 좋아하는 팔로우(Follow) 시스템이라 더욱 흥미로웠고 오랜만에 승부욕도 자극했다.

4주차(04/19 ~ 04/25) - 一難
#

終わらせないで トロイメライ

해냈다
#

  • 김영한 ‘김영한의 실전 데이터베이스 - 기본편’ 완강
  • 『혼자 공부하는 컴퓨터 구조+운영체제』(강민철, 한빛미디어, 2022) 2회독
  • 『하이큐!!』(Haruichi Furudate, 대원씨아이, 2024) 완독
  • 『나는 그대의 책이다』(베르나르 베르베르, 열린책들, 2026) 완독

느꼈다
#

  • 지난주에 이어 ‘데이터베이스 - 기본편’을 완강했다. 서브쿼리, 인덱스가 유독 어려웠다. 사라지지 않은 두려움이 반복으로 극복되면 좋겠다.
  • 한번 더 컴퓨터 구조+운영체제를 학습했다. 지난주에 느낀 감정은 낯선 두려움이었다. 이번에는 읽는 속도도 빨라졌고 이야기가 흥미롭게 느껴졌다.
  • 한 달간의 하이큐 여정이 끝났다. 배구를 전혀 모르지만 속도감이 느껴져 시간 가는 줄 몰랐다. 굶주림을 가장 큰 무기로 삼은 요괴들의 대전쟁은 짜릿하면서 깊은 감동을 남겼다. “좋은 점프는 좋은 도움닫기에서 나온다"는 말처럼, 하루하루를 허투루 보내지 않는 것이 결국 도약의 밑거름이 된다는 걸 다시 한번 새겼다. 하이큐를 읽기 전과 후, 내 삶이 어떤 변화로 연결될지 기대된다.
  • 4 원소로 세계를 구분하고 각각 다른 색지와 폰트를 사용한 독특한 구성이 인상 깊었다. 공기와 흙의 세계는 색지와 폰트의 조합이 잘 어울려 읽는 맛이 있었지만, 고통을 탐구하는 불의 세계는 오히려 조합이 가독성을 떨어뜨려 아쉬웠다. 그럼에도 내면의 성찰을 돕는 독특한 경험 덕분에 언젠가 자녀가 생긴다면 꼭 들려주고 싶다.
  • 오랜만에 한강에 나가 친구들을 만났다. 말을 딱히 하지 않아도 즐거운 분위기가 연출되었다. 푹 쉬었다.

마치며
#

시간이 부족하다는 조급함 속에서도 슬픔과 기쁨을 함께 품는 방법을 배웠다. 손을 뻗을수록 멀어지는 것 같아 좌절도 많았지만, 이대로 끝내고 싶지 않은 소중한 것이라고 느꼈다.

CS와 데이터베이스 앞에서 한없이 작아졌다. 극복할 수 없는 과제처럼 느껴졌는데, 반복하고 나니 생각보다 별거 아니었다. 1주차엔 막막했던 것들이 4주차엔 속도감 있게 읽혔다.

가장 큰 교훈은 하이큐에서 얻었다. 좋은 점프는 좋은 도움닫기에서 나온다. 항상 위를 바라보며 5월로 이어가고 싶다.
